uint8_t WaspLoRaWAN::sendConfirmed ( uint8_t  port,
char *  payload 
)

This function sends a LoRaWAN packet and waits for ACK.

Parameters
char*data: data to be sent uint8_t port: port number to send data
Remarks
data is a sequence of digit representing the value of byte stream expressed in hexadecimal value (i.e.: payload =12A435 – the payload is composed by the following byte stream: 0x12, 0xA4, 0x35 – 6 digit converted in 3 bytes). The maximum length of frame is 584 digit (292 Bytes). User can check _datareceived to know if a downlink was performed
Returns
  • '0' if OK
  • '1' if error
  • '2' if no answer
  • '4' if data length error
  • '5' if error when sending data
  • '6' if module hasn't joined to a network
  • '7' if input port parameter error
