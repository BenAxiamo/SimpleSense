Remarks

uint8_t WaspSigfox::testTransmit ( uint16_t  count,
uint16_t  period,
int  channel 
)

This function is used to send a test SIGFOX frame across a specific channel.

Parameters
uint16_tcount = 0..65535: Count of SIGFOX™ test RF messages. Default: 10
uint16_tperiod = 1..255 Period in seconds between Sigfox test RF messages. Default: 10
intchannel = 0..180 or 220..400 or -1 Use automatic channel selection (default) Channel number to use for SIGFOX™ test RF messages. Channels have a fixed 100 Hz bandwidth, starting at 868.180 MHz for channel 0, ending at 868.198 Mhz for channel 180, restarting at 868.202 MHz for channel 220 and ending at 868.220 MHz for channel 400.
Returns
  • 'SIGFOX_ANSWER_OK' if OK
  • 'SIGFOX_ANSWER_ERROR' if error
  • 'SIGFOX_NO_ANSWER' if no answer
