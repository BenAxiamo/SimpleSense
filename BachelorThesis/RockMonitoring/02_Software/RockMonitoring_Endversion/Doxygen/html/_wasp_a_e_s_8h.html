#define CBC   2

Encryption Mode: In CBC mode, each block of plaintext is XORed with the previous ciphertext block before being encrypted

#define ECB   1

Encryption Mode: The simplest of the encryption modes is the electronic codebook (ECB) mode. The message is divided into blocks, and each block is encrypted separately.

#define PKCS5   3

Padding mode: Padding is in whole bytes. The value of each added byte is the number of bytes that are added, i.e. N bytes, each of value N are added. Example: 01 02 02 03 03 03 04 04 04 04 05 05 05 05 05 etc.

#define ZEROS   4

Padding mode: All the bytes that are required to be padded are padded with zero Example: In the following example the block size is 8 bytes and padding is required for 4 bytes ... | DD DD DD DD DD DD DD DD | DD DD DD DD 00 00 00 00 |
