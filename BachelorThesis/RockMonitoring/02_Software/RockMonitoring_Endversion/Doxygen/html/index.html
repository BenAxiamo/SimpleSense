Rock Monitoring Software Documentation

Project

This is the software documentation of the project Rock Monitoring.
The project is a bachelor thesis from Swen Bischof and Daniel Frei, developped at Bern University of Applied Sciences in 2016.
In the project Rock Monitoring, a prototype was developped to monitor longterm movements of rocks.

Picture of the developped prototype

Software

We used Libelium Waspmote as a development platform.
In this Software documenatation are only the libraries we wrote ourselves.
To be able to upload the code to Libelium Waspmote, the API of Libelium is necessery (see: Libelium Waspmote API )
